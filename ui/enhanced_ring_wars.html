🎯 Ring Wars

Get 3 in a Row to Win!

🏆 Victory Conditions:

  • Same Size Line: 3 rings of identical size in a row, column, or diagonal
  • Size Progression: Small→Medium→Large (or reverse) in any line
  • Concentric Stack: All 3 different sizes stacked in the same cell

⚔️ Battle Rules:

  • 2 warriors take turns placing rings on the 3×3 battlefield
  • Each player commands 3 small, 3 medium, and 3 large rings
  • Rings can be placed on top of different sized rings
  • Same size rings cannot occupy the same position
  • Once placed, rings are locked in battle position
  • If no valid moves remain, skip your turn
